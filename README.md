# dotfiles

Marby's `.dotfiels` Repository with the following features:

- NeoVim as IDE

# Required Software

- NeoVim
- fzf
- ripgrep
- tmux
- lazygit
- k9s

# Installation 

Before you start be sure you will backup your `config` files.

```sh
git clone https://github.com/MarbyRazor/dotfiles.git ~/.dotfiles
./install
```
